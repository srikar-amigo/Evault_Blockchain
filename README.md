This is SIH 2023 project of e-vault based on blockchain.\
It comes under blockchain and cybesecurity domain.\
To run this project we need to:-
  - 1)first install hyperledger fabric setup and then setup couchdb inside of the fabric setup and start the network in Docker.
  - 2)Download and zip the file and replace the application javascript folder in fabric samples by our folder and as well as chaincode-javascript, remaining all files.
  - 3)Up the network and couchdb inside the docker and make sure all the instances are running properly.\
Now you can use our evault on your localserver.
